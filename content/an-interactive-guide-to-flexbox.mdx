## What is Flexbox?

<div class="bg-green-100 dark:bg-green-900 text-green-800 dark:text-green-200 px-4 py-2 rounded-lg shadow-md mb-4">
    <p><strong>Flexbox Definition:</strong> Flexible Box Layout is a CSS module designed for efficient space distribution and powerful alignment capabilities in web layouts.</p>
</div>

Flexbox, short for Flexible Box Layout, revolutionizes the way we design web layouts. It provides a straightforward method to distribute space and align items within a container, regardless of their size or the device's screen resolution.

## Getting Started with Flexbox

To harness the power of Flexbox, you need to understand two key concepts:

1. **Flex Containers**: Elements with `display: flex` or `display: inline-flex`.
2. **Flex Items**: The children elements inside a flex container.

### Flex Container Properties

<div class="bg-yellow-100 dark:bg-yellow-900 text-yellow-800 dark:text-yellow-200 px-4 py-2 rounded-lg shadow-md mb-4">
    <p><strong>Key Flex Container Properties:</strong></p>
    <ul>
        <li>display</li>
        <li>flex-direction</li>
        <li>justify-content</li>
        <li>align-items</li>
        <li>flex-wrap</li>
    </ul>
</div>

Let's explore these properties in detail:

1. **display**: Defines an element as a flex container.
2. **flex-direction**: Sets the main axis for layout (row, column, row-reverse, column-reverse).
3. **justify-content**: Aligns items along the main axis.
4. **align-items**: Controls alignment along the cross axis.
5. **flex-wrap**: Determines if items should wrap to new lines.

### Flex Item Properties

<div class="bg-purple-100 dark:bg-purple-900 text-purple-800 dark:text-purple-200 px-4 py-2 rounded-lg shadow-md mb-4">
    <p><strong>Key Flex Item Properties:</strong></p>
    <ul>
        <li>order</li>
        <li>flex-grow</li>
        <li>flex-shrink</li>
        <li>flex-basis</li>
    </ul>
</div>

These properties give you granular control over individual flex items:

1. **order**: Specifies the order of flex items.
2. **flex-grow**: Defines the ability to grow relative to other items.
3. **flex-shrink**: Determines how items shrink when space is limited.
4. **flex-basis**: Sets the initial main size of an item.

## Interactive Examples and Code Snippets

Let's dive into some practical examples to see Flexbox in action:

### Example 1: Basic Flexbox Container

<div class="bg-gray-100 dark:bg-gray-800 text-gray-800 dark:text-gray-200 px-4 py-2 rounded-lg shadow-md mb-4">
    <p><strong>Try it out:</strong> Copy this CSS into your project to create a centered flex container.</p>
</div>

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
  background-color: #f0f0f0;
}

.item {
  padding: 20px;
  background-color: #3498db;
  color: white;
}
```

This example creates a flex container that centers its child elements both horizontally and vertically.

### Example 2: Responsive Flexbox Grid

<div class="bg-red-100 dark:bg-red-900 text-red-800 dark:text-red-200 px-4 py-2 rounded-lg shadow-md mb-4">
    <p><strong>Pro Tip:</strong> Use this pattern to create responsive grid layouts without media queries!</p>
</div>

```css
.container {
  display: flex;
  flex-wrap: wrap;
  margin: -10px;
}

.item {
  flex-basis: calc(25% - 20px);
  margin: 10px;
  padding: 20px;
  background-color: #2ecc71;
  color: white;
}

@media (max-width: 768px) {
  .item {
    flex-basis: calc(50% - 20px);
  }
}
```

This example creates a responsive grid that adapts from four columns on larger screens to two columns on smaller devices.
